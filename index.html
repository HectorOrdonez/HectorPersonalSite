The exchange rates project

Tech used

Object oriented PHP, Cron Jobs, UML diagrams, MVC Kohana Framework.

The problem

Servers using currency values updating them asynchronously. This was causing differences in the reports.

The solution

Designed, documented and developed system in which the Central Server was the only one updating the currency values and, once the values were updated, they were broadcasted to all other servers.

All possibilities simulation tool

Tech used

Bash, Object oriented PHP, Mocking objets, MVC Kohana Framework.

The problem

The solution

Developed a simulation tool that goes through all possibilities, recording bets, payouts, bonuses and features.
