My name is Hector, and I am an

International Web Developer

Expert in PHP, SQL, JS and CSS.

the skills

I have developed websites, applications and web services in PHP.

Since the old good PHP 5.2 till the newest stable version, 5.5, with which I developed a custom PHP Framework.

With experience in Kohana and Zend Frameworks.

Known and used different design patterns, such as MVC and Factory.

Nowadays mastering the art of TDD (Test Driven Development) with PHPUnit for the unit testings.

I have used SQL as the query language for data management.

Built complex queries with the right use of the different join types.

With practical experience in tables with large amounts of data (>50M of rows) and the importance of the right indexing, column type and storage engine.

As front-end developer, I learned CSS in order to build beautiful websites like this one!

Learned to use CSS to make a dynamic website with cross-browsing styles.

With Responsive Design in mind, I can build websites that adapts to phones, pads and screens alike.

In order to build interactive websites I have used JavaScript to communicate with the user with Ajax and to make beautiful and complex visual effects.

Used object oriented JavaScript in order to build scripts that interact with each other, making light-weighted scripts that, with procedural JavaScript would have been heavy instead.

With experience in the famous JavaScript library jQuery, I have used it in order to simplify or extend the website scripts.

Currently looking for a new employment, a job that challenges me and offers the opportunity to keep developing amazing stuff.
