The exchange rates project

Tech used

Object oriented PHP, Factory pattern, SQL, Cron Jobs, UML diagrams, MVC Kohana Framework.

The problem

Servers using currency values updating them asynchronously. This was causing differences in the reports.

The solution

I designed, documented and developed a system in which one of the servers, called Central Server, was the only one updating the currency values.

All possibilities simulation tool

Tech used

Bash scripting, Object oriented PHP, Mocking objets, MVC Kohana Framework.

The problem

Slot Games random testings provided estimations for total output, but do not provide with maximum payouts or exact averages.

The solution

I developed a simulation tool that was going through all possibilities in a game, recording bets, payouts, bonuses and features.

As there is around 312M possibilities in a Slot Game, one of the challenges was to produce a full simulation in a short time. In order to achieve this I combined PHP with Bash scripting building a multi-thread simulation system, capable to reduce the simulation time cost in more than 8 times.

The building location register project

Tech used

PHP5, SQL, JavaScript, jQuery, Google Maps Api.

The problem

A feature requested for a website, which had buildings registered around the world, to allow Users to set the specific building location in Google Maps.

Data Importers and Exporters

Tech used

Object oriented PHP, Kohana Framework, JSON, XML.

The problem

Data in different formats required to be imported to the system's database.

This data eventually generated reports, which had to be exported in different formats depending on User's choice.

The solution

Implemented several features that included the following characteristics.

  • JSON and XML Data validation. The files imported by User had to be validated before adding them to the database.
  • Dynamic SQL queries. The imported data had to be turned dynamically into SQL constructions.
  • XLS and PDF generators. The export tools had to make use of the XLS and PDF generators in PHP to compile data in a readable format.
